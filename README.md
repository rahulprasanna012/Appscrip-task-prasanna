Setup Instructions
Create Next.js Project:

bash
npx create-next-app@latest ecommerce-store
cd ecommerce-store
Select options during setup:

TypeScript: No

ESLint: Yes

Tailwind CSS: No

src/ directory: No

App Router: Yes

Copy all files from this structure into your project

Install dependencies:

bash
npm install
Run development server:

bash
npm run dev
Open browser:
Navigate to http://localhost:3000

Features Implemented
✅ Product Display - Fetches from fakestoreapi.com
✅ Add to Cart - With quantity controls
✅ Wishlist - Add/remove products
✅ Filters - By category and customization
✅ Sorting - Price, newest first
✅ Responsive Design - Mobile, tablet, desktop
✅ Cart Panel - Slide-in panel with total
✅ Wishlist Panel - View and manage wishlist

API Used
https://fakestoreapi.com/products - All products

https://fakestoreapi.com/products/categories - Categories
